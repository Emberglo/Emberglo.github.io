Keepr

View This Project

A solo build application designed to replicate the premise of pinterest. Users can view all keeps submitted to the site, and logged in users can create their own keeps and create vaults to collect their and other users keeps.
This project was built using HTML, CSS, JavaScript, VueJS on the front end, C# with .Net for the API, and MySQL for the server. It also uses Auth0 for user authentication.

Bar Trivia Management Application

View This Project

For this project I collaborated with a team of three other developers to plan and build a full stack application for managing bar trivia games for hosts and players. This Saas application is intended to be marketed to local trivia companies to replace apps that are currently in use but have a poor customer experience. My primary contribution was to the front end of the application but I was involved throughout the build process.
This project was built using HTML, CSS, Sass, JavaScript, and VueJS on the front end, NodeJS with Express for the API, and MongoDB with Mongoose for the server. It also uses Auth0 for user authentication.
